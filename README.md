# tidal-cli-music-player
tidal-cli-music-player is a terminal music player for TIDAL with a daemon/client design so playback persists across commands. It streams real lossless/hi-res audio via mpv, with a full-screen TUI and scriptable CLI: search, browse, mixes, favorites, time-synced lyrics, terminal cover art (Sixel/Kitty), and MPRIS media-key support.
